# Portfólio Pessoal Responsivo

## Breve Descrição do Portfólio
Este projeto consiste no desenvolvimento de um portfólio pessoal responsivo, simulando a página profissional de um desenvolvedor. O site apresenta informações pessoais, habilidades técnicas (Front-end e Back-end), vitrine de projetos e um formulário de contato. O layout foi construído inteiramente com HTML semântico e CSS puro, garantindo boa performance e adaptação a diferentes tamanhos de tela (mobile e desktop) sem o uso de frameworks.

## Principais Desafios Encontrados
* **Gestão de Tempo (Trabalho Individual):** Planejar e executar todo o ciclo do projeto (HTML, CSS, Deploy e Vídeo) sozinho.
* **Responsividade com CSS Puro:** Ajustar o empilhamento dos cards de projetos e adaptar o menu de navegação para telas de celulares utilizando estritamente Flexbox e Media Queries, sem o auxílio de ferramentas como Bootstrap ou Tailwind.
* **Design e Identidade Visual:** Criar um tema escuro (Dark Mode) do zero, garantindo um contraste elegante e leitura confortável para o usuário.

## O Que Cada Integrante Fez
Como o projeto foi desenvolvido individualmente, fui o único responsável por 100% das etapas:
* Estruturação semântica do HTML (header, main, section, articles, footer).
* Estilização visual avançada com CSS (Dark Mode, layout em Flexbox, botões estilizados com efeitos hover).
* Implementação da responsividade e adaptação de imagens para dispositivos móveis.
* Configuração do repositório no GitHub, versionamento via commits e deploy no GitHub Pages.
* Gravação do vídeo de apresentação do código e do layout final.
